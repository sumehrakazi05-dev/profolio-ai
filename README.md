### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (distributed with Node.js)

### Installation

1. **Navigate to the project directory**
   ```bash
   cd portfolio-generator
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the server**
